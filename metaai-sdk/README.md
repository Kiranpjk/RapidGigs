# Meta AI SDK

A lightweight Python SDK + FastAPI server that reverse‚Äëengineers the Meta AI website (`meta.ai`) to provide image and video generation using only the browser cookies for authentication‚Äîno official Meta API keys required.

## Quick Start

1. **Install dependencies**  
   ```bash
   cd metaai-sdk
   python -m venv .venv && source .venv/bin/activate
   pip install -r requirements.txt   # generated by pyproject.toml
   ```

2. **Set up environment variables**  
   Copy the example and fill in your cookies:

   ```bash
   cp .env.example .env
   # Edit .env and add:
   # META_AI_DATR=...
   # META_AI_ABRA_SESS=...
   # META_AI_ECTO_1_SESS=...
   ```

3. **Capture the `doc_id` values**  
   * See `refresh_guide.md` for step‚Äëby‚Äëstep instructions on extracting the `doc_id` for image and video generation from your browser.

4. **Update the SDK**  
   Replace the placeholder `DOC_ID` constants in `src/metaai_sdk/image.py` and `src/metaai_sdk/video.py` with the captured values.

5. **Run the FastAPI server**  
   ```bash
   uvicorn api.server:app --reload
   ```

6. **Test the endpoints**  
   - Health check: `GET http://localhost:8000/healthz` ‚Üí `{"status":"ok"}`
   - Generate an image: `POST http://localhost:8000/image` with JSON `{ "prompt": "A sunrise over mountains" }`
   - Generate a video: `POST http://localhost:8000/video` (or `/video/async` then poll `/video/jobs/{job_id}`)

## Detailed API

| Method | Path | Request Body | Response |
|--------|------|--------------|----------|
| `GET` | `/healthz` | ‚Äì | `{"status":"ok"}` |
| `POST` | `/image` | `{ "prompt": "<text>", "orientation":"LANDSCAPE|VERTICAL|SQUARE" }` | `{ "success": true, "image_urls": [...], "prompt": "<text>" }` |
